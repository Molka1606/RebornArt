## Introduction
My name is **Molka**, and I developed the **User module** as part of the RebornArt web project.  
RebornArt is a web platform dedicated to creative recycling and artistic reuse. The goal of this module is to manage users and ensure secure access to the platform.  
This module represents a fundamental component of the system, as it handles authentication, authorization, and user interaction.

---

## Project Objectives
The main objectives of this module are:
- To manage user accounts  
- To ensure secure authentication  
- To handle user sessions correctly  
- To provide a clear separation between user and admin roles  

---

## Implemented Features
In this module, I implemented the following features:
- User registration  
- User login and logout  
- Session management  
- Profile access  
- Role-based access control (User / Admin)  
- Secure navigation between pages  

---

## Technologies Used
The User module was developed using:
- PHP with MVC architecture  
- MySQL database  
- HTML, CSS, and JavaScript  
- XAMPP (Apache and MySQL)  
- Git and GitHub for version control
